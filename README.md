# 🎨 AI Image Generator

An AI-powered image generation app built with **React Native**, **Expo SDK 54**, and **Gemini AI**. Users can enhance prompts with AI, generate stunning images, preview them, save generation history, and share creations.

---

---

# ✨ Features

- 🤖 AI Prompt Enhancement using Gemini AI
- 🎨 AI Image Generation
- 🖼️ Full Screen Image Preview
- 🕒 History of Generated Images
- 💾 Local Storage using AsyncStorage
- 📤 Share Generated Images
- 🎭 Multiple Image Styles
- 📱 Modern UI with Expo Router
- ⚙️ Settings Screen
- 🗑️ Clear History

---

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/ayush08m/ImageGenerator-AI.git
```

Move into the project

```bash
cd AiImageGenerator
```

Install dependencies

```bash
npm install
```

Start Expo

```bash
npx expo start
```

---

# 🔑 Environment Variables

Create a `.env` file in the project root.

```env
EXPO_PUBLIC_GEMINI_API_KEY=YOUR_API_KEY
```

---

# 📖 How It Works

1. Enter a text prompt.
2. Enhance the prompt using Gemini AI (optional).
3. Choose an image style.
4. Generate an AI image.
5. Preview the generated image.
6. Share the image.
7. View previous generations in the History screen.
